# Data-Masking-API-Custom-Policy

Fields Masking Policy enhances the data security and privacy by applying Masking to the sensitive fields information such as SSN, Address, Payroll, etc.

Masks/Protects the value of defined fields available in the Response payload to ensure its not being exposed to the client applications.

This Custom policy supports masking for the APIs that return the response payload in the following data formats.
 -> XML
 -> JSON
 -> CSV
 
This Custom policy provides cushion to the developers with couple of masking options as an added feature.
 -> Generic Masking -> Masks field values anywhere and everywhere.
 -> Path Based Masking -> Masks field values at the defined payload level/hierarchy.
